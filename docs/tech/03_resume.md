# 職務経歴書：フルスタック・ビジネスアーキテクト（2026年4月時点）

## ■ 職種
経営コンサルタント / フルスタックエンジニア / プロダクトマネージャー

## ■ 専門性・技術スタック
「ビジネスの収益性を技術で担保し、システムコストを最小化させるアーキテクチャ設計」を専門としています。

### 1. 開発言語・フレームワーク
- **TypeScript**: 静的型付けによる堅牢なフロントエンド/バックエンド共通開発。
- **Hono (Node.js/Cloudflare Workers)**: 2026年現在の最速かつ軽量なサーバーレスフレームワークの選定。
- **HTMX**: 過剰なJavaScript（SPA）を排除し、サーバーサイド駆動で高速かつ低コストなUI/UXを実現する「原点回帰」の設計。
- **JSX (Hono/jsx)**: コンポーネント志向を維持しつつ、ビルドステップを簡略化したUI実装。

### 2. インフラ・データベース
- **Cloudflare Ecosystem**: Workers, D1, KV, R2を活用したエッジコンピューティング環境の構築。
- **SQLite / D1**: サーバーレス環境でのトランザクション管理とコスト最適化。
- **SQL (Query Builderレス)**: ORMのオーバーヘッドを避け、生SQLによるパフォーマンス最大化。

### 3. 特化スキル
- **ビジネス・オートメーション**: Playwrightを用いた市場モニタリング、Chrome拡張機能による業務自動化。
- **フィナンシャル・エンジニアリング**: freee等の会計API連携、融資・利息計算ロジックの実装。
- **AIネイティブ開発**: Gemini Live等を活用し、AIと協働して要件定義から実装までを一人で完結させる「1人開発チーム」の運用。

## ■ プロジェクト実績（抜粋）

### A. サービス予約・管理プラットフォームの構築
- **概要**: 経営コンサルティングの知見を活かし、小規模店舗・セラピスト向けの予約システムを開発。
- **役割**: 要件定義、アーキテクチャ設計、実装、運用設計。
- **技術選定の意図**: 
    - D1を活用し、1ユーザーあたりのデータ蓄積コストを最小化（無料枠〜低額プランでの利益率最大化）。
    - HTMXによる「ドリルダウン検索」の実装により、スマホユーザーの「思考を止めない」UXを高速に実現。
    - PWA（Progressive Web Apps）化を前提とした設計で、メール配信コスト（API費用）を削減。

### B. クライアント「K」財務モニタリングシステム
- **概要**: 500万円規模の融資・返済管理およびfreee会計ソフトのオンボーディング。
- **成果**: 
    - 複雑なCSVデータから元本・利息を自動判別し、会計ソフトへのインポート手順をマニュアル化。
    - 経営判断に直結するリアルタイムなモニタリング体制を構築。

## ■ ビジネス視点（コンサルタントとしての強み）
- **ROI最適化**: 外部API（SendGrid, Resend等）の利用を必要最小限に抑え、LTV（顧客生涯価値）を高めるための低コスト運営を徹底。
- **ドメイン知識**: 会計、融資、法務要件を直接コードに落とし込めるため、エンジニアとビジネスサイドの「翻訳コスト」をゼロ化。
- **グローバル対応**: 英語の「リズムベース聴解法」を理解し、英語圏の最新ドキュメントやグローバルチームとの協働が可能。

## ■ 自己PR
私は「コードが書けるコンサルタント」ではなく、「技術を用いて事業の不確実性を排除するアーキテクト」です。
2026年現在のAIがコードを書く時代において、重要なのは「何を、どの構成で、いくらのコストで作るか」という意思決定です。経営コンサルタントとして培った「数字」への責任感と、最新の軽量サーバーレススタックを用いた「実装力」を掛け合わせ、最短距離で利益を生むプロダクトを構築します。
